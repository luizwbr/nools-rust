# Node.js Examples for nools-rust

These examples demonstrate how to use nools-rust from Node.js.

## Setup

```bash
npm install
```

## Running Examples

### JavaScript Example
```bash
npm run hello
```

### TypeScript Example
```bash
npm run counter
```

## Examples

- `hello_world.js` - Basic rule matching with messages
- `counter.ts` - Counter example with TypeScript types

## Note

These examples require the native bindings to be built. See the parent README for build instructions.

To build the native module:
```bash
cd ../..
npm install
npm run build
```
